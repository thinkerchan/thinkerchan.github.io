常用curl命令

读取网页

$ curl https://baidu.com

保存文件

$ curl https://baidu.com > baidu.html

或者

$ curl https://baidu.com -o baidu.html

代理模式

$ curl -x 123.45.67.89:1080 https://baidu.com > baidu.html

存储cookie

$ curl https://baidu.com > baidu.html -D cookie.txt

使用cookie

$ curl https://baidu.com > baidu.html -D cookie.txt -b readcookie.txt

添加UserAgent

$ curl -A “Mozilla/4.0 (compatible; MSIE 6.0; Windows NT 5.0)” https://www.baidu.com

添加referer

$ curl -A “Mozilla/4.0 (compatible; MSIE 6.0; Windows NT 5.0)” -e “index.baidu.com” https://www.baidu.com
